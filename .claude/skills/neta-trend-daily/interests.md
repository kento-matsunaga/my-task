# 興味プロファイル

このファイルはスキル実行時に参照される。ユーザーの興味に合わせてカスタマイズすること。

## 興味領域（優先度順）

### Tier 1: 最高興味（★5候補）
- **Claude Code**: 最新の使い方、Tips、アップデート情報、拡張機能、Skills、MCP
- **バイブコーディング / Vibe Coding**: AIとの対話的な開発スタイル、プロンプト駆動開発
- **AIエージェント**: 自律型エージェント、マルチエージェント、Agentic Workflow

### Tier 2: 高興味（★4候補）
- **AI全般**: LLM、生成AI、RAG、ファインチューニング、新モデルリリース
- **Go言語**: 新機能、ライブラリ、パフォーマンス、設計パターン
- **Webサービス作成**: SaaS、個人開発、インディーハッカー、収益化

### Tier 3: 中興味（★3候補）
- **面白ネタ / Fun要素**: 技術の意外な活用法、ユニークなプロジェクト、Show HN系
- **開発者ツール**: エディタ、ターミナル、CLI、DevOps
- **フロントエンド / UI**: React, Next.js, Tailwind, デザインシステム

### Tier 4: 低興味（★2候補）
- **インフラ / クラウド**: AWS, GCP, Kubernetes, Docker（直接的でなければ低め）
- **セキュリティ**: 脆弱性情報、セキュリティツール
- **プログラミング言語（Go以外）**: Rust, Python, TypeScript の新機能等

### Tier 5: 興味外（★1候補）
- **企業ニュース**: 資金調達、M&A、人事（技術的内容が薄い場合）
- **ハードウェア / 半導体**: チップ設計、デバイス（ソフトウェアと無関係な場合）
- **政治・規制**: AI規制、法律（技術的影響が薄い場合）

## 評価のガイドライン

- 複数の興味領域にまたがる記事は、最も高い Tier で評価する
- 例: 「Claude Codeで Go の Web API を自動生成」→ Tier1 (Claude Code) × Tier2 (Go) × Tier2 (Web) = ★★★★★
- 記事のブクマ数/Pointsが異常に高い場合（上位3位以内）、+0.5の補正を考慮する（ただし★5上限）
- 「面白い」「バズっている」「予想外」な記事は面白ネタ補正で+1を検討する
